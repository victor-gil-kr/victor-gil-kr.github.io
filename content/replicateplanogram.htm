ReplicatePlanogram

This command creates up to 10 copies of the selected planogram, with varying widths for each new planogram. You must select a planogram before using this action in the script.

Syntax

Visual Basic

Copy 
ReplicatePlanogram ( _
    Planograms As Integer, _
    Length0 As Double, _
    Length1 As Double, _
    Length2 As Double, _
    Length3 As Double, _
    Length4 As Double, _
    Length5 As Double, _
    Length6 As Double, _
    Length7 As Double, _
    Length8 As Double, _
    Length9 As Double, _
    StretchSegments As Boolean, _
    CopyPositions As Boolean, _
    CopyPerformances As Boolean, _
    CopyDrawings As Boolean _
)

C#

Copy 
ReplicatePlanogram(int Planograms, [double Length0 = 0], [double Length1 = 0], [double Length2 = 0], [double Length3 = 0], [double Length4 = 0], [double Length5 = 0], [double Length6 = 0], [double Length7 = 0], [double Length8 = 0], [double Length9 = 0], [bool StretchSegments = false], [bool CopyPositions = false], [bool CopyPerformances = false], [bool CopyDrawings = false])

Parameters

Parameter

Description
Planograms

Integer. Number of planogram copies to be created.
Length0-9 (optional)

Double. Widths of the planogram copies to be created. The number of Length values specified must correspond to the number of planogram copies being created. Click the remaining Length headings to disable the fields.

StretchSegments (optional)

Boolean. Specifies whether to add or remove fixtures and segments (type "0"), or to stretch existing fixtures and segments (type "1"), to fill the space in the new planograms.

  • The Add/Remove option is useful when you build your planograms using consistent segment and fixture widths to represent modular fixture components.
  • The Stretch option is useful when you build your planograms using segments and fixtures with widths that vary or that extend past the modular segments.

CopyPositions (optional)

Boolean. Type "1" to copy positions from an existing planogram to the new planograms.

CopyPerformances (optional)

Boolean. Type "1" to copy Performance-level data from the existing planogram to the new planograms.

CopyDrawings (optional)

Boolean. Type "1" to copy text boxes and other drawing objects from the existing planogram to the new planograms.

Example: Replicate a planogram

Classic style scripting

Visual Basic

Copy 
SpacePlanning.OpenProjectFile("C:\Program Files (x86)\JDA\Intactix\Samples\Space Planning Planograms\Sample - Coffee Makers.psa")
SpacePlanning.SelectPlanogram("12' Coffee Makers", True)
SpacePlanning.ReplicatePlanogram(2, 12, 14)

C#

Copy 
SpacePlanning.OpenProjectFile(@"C:\Program Files (x86)\JDA\Intactix\Samples\Space Planning Planograms\Sample - Coffee Makers.psa");
SpacePlanning.SelectPlanogram("12' Coffee Makers", true);
SpacePlanning.ReplicatePlanogram(2, 12, 14);

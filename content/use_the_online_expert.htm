Use the OnLine Expert

Search for content

You can use the Search field in the OnLine Expert's header to search for content.

You can use the following options to search for words or phrases in the Search field:

Search object

Example

Result

A single word

Select

Topics that include the word "select", including variations such as "selector" and "selection"

A phrase

"new operator"

Topics that include the literal phrase "new operator" and grammatical variations such as "new operators"

new operator

Topics that include the words "new" and "operator" in addition to the phrase "new operator"

Multiple terms in the same topic

ship rate

ship AND rate

ship & rate

ship + rate

Topics that include both the words "ship" and "rate" as well as variations of the words, such as "shipping" and "rated"

Separate terms in a topic

domestic OR foreign

domestic | foreign

Topics that include either the word "domestic" or the word "foreign" or both words

One term without another term

product NOT regular

product ! regular

Topics that include the word "product" but not the word "regular"

The following rules apply when you search:

  • Searches are not case-sensitive, so you can enter your search in uppercase or lowercase characters.
  • You can search for any combination of letters (a–z) and numbers (0–9). However, a minimum of three characters must be provided to return a result.
  • The default operator for multiple word search expressions is AND. For example, if you enter Main Menu as a search term, you create the logical equivalent to "main AND menu" for the search.

When you submit a search request, the search results are displayed in the main content window. You can click a topic's heading to display the topic.

Print a topic

You can print a topic to a printer or save it to a PDF file.

  1. Search for content in the OnLine Expert.
  2. Select the topic to display and print.

  3. If the topic contains expanding sections, click to expand all sections in the topic.

    Note: You also can click to expand the individual topics to be included in the print output.

  4. Click . The system's print controls are displayed.
  5. Use the print controls to print the topic or save it to a PDF file.

Display the navigation pane

The OnLine Expert uses responsive technology, which means that the help window supports both full-sized and condensed displays for optimal viewing on desktop, tablet, and mobile devices. When the OnLine Expert window is condensed (such as when it opens in a floating window or is viewed on a tablet or mobile device), the navigation pane is hidden to provide a larger view of the help content.

To access the table of contents and search function on the navigation pane, click the menu icon. If your device supports a larger window, you can resize or maximize the help window to display the navigation pane.

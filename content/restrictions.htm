Restrictions

The following guidelines should be used when you create Space Automation Professional scripts:

Configuration field adjustment

There is a different process to access Application (Configuration) fields. Configuration object fields are accessed through SpacePlanning.Application and FloorPlanning.Application statements instead of using the Get<Object>Field or Set<Object>Field commands that are available for other objects.

CKS script notes

  • CKS assemblies are run through the Category Batch Processing middle tier in invisible unattended mode. Therefore, they should not contain commands that display a user interface. If the CKS assembly script requires a response or displays a user interface, the script will time out and be marked as a failed CKS job.
  • Floor Planning commands should not be added to a CKS assembly or to a script assembly that is associated with Space Planning.

  • In a CKS script, you cannot use the Space Planning Automation add-on pattern to take over or block Space Planning commands and events by using Notify. See "Implement an in process add-on COM component" in the Space Planning Automation Help for more information.

Validation script notes

  • In a validation script, you cannot use the Space Planning Automation add-on pattern to take over or block Space Planning commands and events by using Notify. See "Implement an in process add-on COM component" in the Space Planning Automation Help for more information.

In-process assemblies

If you create an assembly that will be used in-process with Space Planning (for example, a CKS or Space script assembly), you must ensure that the script assembly contains only one class that implements the script interface. If more than one class exists in the script assembly that implements the script interface, an execution error can occur because Space Planning cannot determine the class to use.

For Each vs. For loop

If you use a For Each loop to change segment properties at run time, an error may occur. To avoid this situation, use a For loop instead.

Copy 
Dim Plano as Space.Planogram
Dim Factor As Double = 1 / 2.54
Plano= SpacePlanning.ActivePlanogram
For index = 0 To Plano.Segments.Count - 1
    Dim Seg As Space.Segment
    Seg = Plano.Segments.Item(index)
    On Error Resume Next
    'Change Segment dimensions
    tempVal = Seg.GetSegmentField(SpaceSegmentFields.Width)
    Seg.SetSegmentField(SpaceSegmentFields.Width, (tempVal * Factor))
    tempVal = Seg.GetSegmentField(SpaceSegmentFields.Height)
    Seg.SetSegmentField(SpaceSegmentFields.Height, (tempVal * Factor))
    'Change Door Frame dimensions
    tempVal = Seg.GetSegmentField(SpaceSegmentFields.FrameWidth)
    Seg.SetSegmentField(SpaceSegmentFields.FrameWidth, (tempVal * Factor))
    tempVal = Seg.GetSegmentField(SpaceSegmentFields.FrameHeight)
    Seg.SetSegmentField(SpaceSegmentFields.FrameWidth, (tempVal * Factor))
Next 'Segment

Configuration settings to launch Space Planning or Floor Planning

If you deselect the options to launch Space Planning or Floor Planning in the Configuration dialog box, Space Automation Professional scripts will fail with an exception, "Object reference not set to an instance."

Exceptions are best avoided if Space Planning and Floor Planning instances are created before the script is executed. The script file does not have control over script lines that are not part of the Class Library. If the Configuration dialog box stops to create an instance, the Space Automation Professional code does not have control on these objects. It is your responsibility to handle these exceptions. This error message will be logged into the log file at the specified log file path.

You can add the following lines of code to scripts for better exception handling in these situations based on the application launch check for Space Planning and Floor Planning.

Visual Basic

Copy 
'If the Space Planning launch option is not selected:
If (SpacePlanning Is Nothing) Then
    Throw New Exception("Space Planning has not been started. Launch the application before you execute the script.")
End If
'Script lines go here.
'If the Floor Planning launch option is not selected:
If (FloorPlanning Is Nothing) Then
    Throw New Exception("Floor Planning has not been started. Launch the application before you execute the script.")
End If
'Script lines go here.
'If either or both options are not selected:
If ( (SpacePlanning Is Nothing) Or (FloorPlanning Is Nothing)) Then
    Throw New Exception("Space Planning or Floor Planning has not been started.")
End If
'Script lines go here.

C#

Copy 
if(SpacePlanning == null)
    throw new Exception("Space Planning has not been started. Launch the application before you execute the script.")
    //Script lines go here.
if(FloorPlanning == null)
    throw new Exception("Floor Planning has not been started. Launch the application before you execute the script.")
    //Script lines go here.
if(SpacePlanning == null || FloorPlanning == null)
    throw new Exception("Space Planning or Floor Planning has not been started.")
    //Script lines go here.

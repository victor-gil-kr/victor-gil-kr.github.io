DeleteDrawingsWithCondition

This command enables you to remove all drawings that meet a designated condition from the active floorplan. This command enables you to delete drawing objects without embedding the DeleteDrawing command in a ForDrawings loop.

Syntax

Visual Basic

Copy 
DeleteDrawingsWithCondition ( _
    WhereCondition As String _
)

C#

Copy 
DeleteDrawingsWithCondition(string WhereCondition)

Parameters

Parameter

Description
WhereCondition

String. The criteria used to filter which objects are deleted by the command.

The following operators are supported:

  • = (Equal)
  • != or <> (NotEqual)
  • > (GreaterThan)
  • < (LessThan)
  • >= (GreaterThanOrEqual)
  • <= (LessThanOrEqual)
  • BETWEEN: Supported for numeric and date data only

  • LIKE: Supported for string data only. This operator can be used with wildcards:

    • Name.LIKE(CUP%): Searches for a string with CUP as the starting characters

    • Name.LIKE (%CUP): Searches for a string with CUP as the characters

    • Name.LIKE (%CUP%): Searches for CUP anywhere within the string

  • IN: Supported for string, numeric, and enumerated data only
  • AND

  • OR

  • NOT

When strings are provided, the following text delimiters are supported:

Note: The delimiters are required if the search string contains "and" or "or".

  • No delimiter (C# and VB): Application.Command ("Planogram.Name = Sample POG")
  • \" (C# only): Application.Command ("Floorplan.Name=\"Sample and Test\");
  • "" (VB only): Application.Command ("Floorplan.Name=""Sample or Test""")

Notes:

  • For operators that support date values, the date format must be MM-DD-YYYY.

  • Searches on fields with enumerated data must refer to the defined enumerated string. The enumeration constant cannot be used. For example:

    • Application.Command ("PackageStyle=BOX") is acceptable.

    •  Application.Command ("PackageStyle=0") fails.

The condition can access parent objects by including a reference to the object. The condition can access child objects by including the Any operator. See the example script for the appropriate examples.

Example: Delete multiple objects on a floorplan

Classic style scripting

Visual Basic

Copy 
FloorPlanning.OpenProjectFile("C:\Program Files (x86)\JDA\Intactix\Samples\Floor Planning Floorplans\Sample - Store 001.PFA")
FloorPlanning.DeleteDepartmentsWithCondition("Name = Housewares")
FloorPlanning.DeleteDrawingsWithCondition("Name = SampleDrawing")
FloorPlanning.DeleteFixturesWithCondition("Name = 0060(SODA CASE)")
FloorPlanning.DeletePlanogramsWithCondition("Name = 22' Beer")
FloorPlanning.DeleteSectionsWithCondition("Key = 012160")
FloorPlanning.DeleteFloorplansWithCondition("Name = 001 Sample Store")

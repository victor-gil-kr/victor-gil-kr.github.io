CreateDatabaseReportPreFilter

This command enables you to create a pre-filter for use with the ExportDatabaseReportData command. The pre-filter is dynamic, so it is not saved to the database.

Pre-filters enable you to limit the data displayed that is exported in the report. If a pre-filter is not applied to the database before exporting the report, all associated data in the database is exported, which might make the report difficult to view.

Notes: 

  • You must include the SelectDatabase command before this command to select the database to be accessed by your script.

  • This command is not recommended in Category Knowledge Server scripts.

  • This command is supported only when Space Automation Professional is configured to use a direct database connection.

Syntax

Visual Basic

Copy 
Dim Var As PreFilterConditions = New PreFilterConditions()
CreateDatabaseReportPreFilter(_
    PreFilterName As String, _
    PreFilterConditions As PreFilterConditions _
)

C#

Copy 
PreFilterConditions Var = new PreFilterConditions();
CreateDatabaseReportPreFilter(string PreFilterName, PreFilterConditions PreFilterConditions)

Parameters

Parameter

Description
PreFilterName

String. The CKB pre-filter to be created to limit the data in the analysis report or chart.
PreFilterConditions

PreFilterConditions. A type that holds the values for State, Object, Field, Operator, and Value. You can specify up to five filters( Filter0 – Filter4).
StateX

FilterState. The status to enable pre-filter parameters for selection. State0 is set to ON, but you can set State1-4 to OFF, AND, or OR. The Off option disables the filter options for the associated row. The AND and OR options enable you to specify additional filter criteria.

ObjectX

The object by which you want to filter the CKB report data. You can specify up to five objects (Obj0-Obj4) by which to filter report data.

FieldX

The field you want to include in the CKB report pre-filter. You can specify up to five fields (Fld0-Fld4) by which to filter CKB report data.

OperatorX

The operator specifying the condition included in the CKB report pre-filter. You can specify operators for each field used to filter the data.

ValueX

The value being compared to filter the CKB report data. You can specify values for each field used to filter the data.

Example: Export database report data using a pre-filter

Classic style scripting

Visual Basic

Copy 
'Select Database
SpacePlanning.SelectDatabase("CKB_9.0 | (LOCAL) | SA ")
'Create PrefilterConditions object 
Dim myConditions As PreFilterConditions = New PreFilterConditions()
'Assign values to filters 
'ON --- First filter state should be On (True)
myConditions.State0 = True
myConditions.Object0 = "Space Planogram"
myConditions.Field0 = "Width"
myConditions.Operator0 = [Operator].GREATER_THAN
myConditions.Value0 = "100"
'Continue if more filter conditions are required
'Filter1 ON
myConditions.State1 = FilterState.AND
myConditions.Object1 = "Space Planogram"
myConditions.Field1 = "Height"
myConditions.Operator1 = [Operator].GREATER_THAN
myConditions.Value1 = "12"
'Create PreFilter
Database.CreateDatabaseReportPreFilter("TestFilter", myConditions)
'Export data for created prefilter
Database.ExportDatabaseReportData("TestReport", "dbo", "C:\Temp\Width_Planogram_Report.htm", ExportMethod.HTML, True, False, True, , ,"TestFilter")

C#

Copy 
//Select Database
SpacePlanning.SelectDatabase("CKB_9.0 | (LOCAL) | SA ");
//Create PrefilterConditions object
PreFilterConditions myConditions = new PreFilterConditions();
//Assign values to filters ON --- First filter state should be On (True)
myConditions.State0 = true;
myConditions.Object0 = "Space Planogram";
myConditions.Field0 = "Width";
myConditions.Operator0 = ">";
myConditions.Value0 = "100";
//Continue if more filter conditions are required
//Filter1 ON
myConditions.State1 = FilterState.AND;
myConditions.Object1 = "Space Planogram";
myConditions.Field1 = "Height";
myConditions.Operator1 = ">";
myConditions.Value1 = "12";
//Create PreFilter
Database.CreateDatabaseReportPreFilter("TestFilter", myConditions);
//Export data for created prefilter
Database.ExportDatabaseReportData("TestReport", "dbo", @"C:\Temp\Width_Planogram_Report.htm", ExportMethod.HTML, true, false, true,””,””,"TestFilter");

SaveScriptDocument

This command enables you to save information as a document in the ix_net_iks_doc table in the Category Knowledge Base database. The document could be a text-based file, PDF document, or an uploaded database planogram that is made available as a .psa file. After the document is saved to the table, it can be downloaded and viewed in the appropriate document viewer. Browser-supported documents can also be viewed directly in the web application.

Note: If you create temporary files that are saved as Category Knowledge Server documents to the database, you may want to delete those files after they are saved to the database to prevent the accumulation of an excessive number of temporary files.

Note: This command is supported only when Space Automation Professional is configured to use a direct database connection.

Syntax

Visual Basic

Copy 
SaveScriptDocument ( _
    DefaultFileName As String, _
    Description As String, _
    FileType As String, _
    ContentType As String, _
    Filename As String _
)

C#

Copy 
SaveScriptDocument(string DefaultFileName, string Description, string FileType, [string ContentType = ""], [string FileName = ""])

Parameters

Parameter

Description
DefaultFileName

String. The name to be displayed for the CKS output document in Open Access. This value is also the default file name that is used when a remote user downloads and saves the document on their machine.
Description

String. The description to be displayed for the CKS output document in Open Access.

FileType

String. The optional file type that is displayed in Open Access to identify the document type for the remote user. Default options are Space Planning Project File (displays PSA), PDF Document (displays PDF), or Text File (displays TXT).

ContentType (optional)

String. The optional document type or MIME (Multipurpose Internet Mail Extensions) type associated with the document. This identifier is used by the web browser to determine the client application that opens the document. If ContentType is blank, the web browser attempts to determine the client application from the file extension of the DefaultFileName. If both values are blank or do not match an application installed on the remote system, the remote user can only save (not open) the document. Common ContentType values include: application/psa (for a Space Planning planogram), text/HTML (for an HTML document), text/plain (for a text document), and application/pdf (for an Adobe Acrobat PDF document).

FileName (optional)

The path and file name of the planogram or document to be saved to the ix_net_iks_docs table in the Category Knowledge Base database. If the path and file name are not specified, the active project is saved to the database and made available for download.

Example: Open a CKS object from the database

Classic style scripting

Visual Basic

Copy 
'Note: Set PDF Writer to save generated PDF documents to the C drive.
Try
    Dim sFilePath, sName As String
    Dim iCount As Integer = 0
        SpacePlanning.OpenScriptObjectsFromDatabase(True, FileOpen.SingleProject)
        For Each Plano As Space.Planogram In SpacePlanning.ForPlanograms
            sFilePath = "C:\" & Plano.GetPlanogramField(SpacePlanogramFields.Name).ToString & ".PDF"
                If System.IO.File.Exists(sFilePath) Then
                    System.IO.File.Delete(sFilePath)
                End If
                SpacePlanning.SetActivePlanogram(iCount)
                sName = Plano.GetPlanogramField(SpacePlanogramFields.Name).ToString & ".PDF"
                SpacePlanning.PrintUsingLayout("C:\Program Files (x86)\JDA\Intactix\Intactix Knowledge Base\Layout\Front View.psy", SpaceSelection.CurrentPlanogram, "JDA PDF Writer", sFilePath)
                SpacePlanning.SaveScriptDocument(sName, Plano.GetPlanogramField(SpacePlanogramFields.Name).ToString & "PDF DOCUMENT", "PDF Document", "application/pdf", sFilePath)
                iCount = iCount + 1
            Next
        Catch ex As Exception
        SpacePlanning.AddToMessageList(1000, ex.StackTrace, MessageSeverity.Error)
        SpacePlanning.AddToMessageList(1000, ex.Message, MessageSeverity.Error)
End Try

C#

Copy 
//Note: Set PDF Writer to save generated PDF documents to the C drive.
try
    {
    string sFilePath, sName;
    int iCount = 0;
    SpacePlanning.OpenScriptObjectsFromDatabase(true, FileOpen.SingleProject);
    foreach(Space.Planogram Plano in SpacePlanning.ForPlanograms())
        {
        sFilePath = @"C:\" + Plano.GetPlanogramField(SpacePlanogramFields.Name).ToString + ".PDF";
        if (System.IO.File.Exists(sFilePath))
        System.IO.File.Delete(sFilePath);
        SpacePlanning.SetActivePlanogram(iCount);
        sName = Plano.GetPlanogramField(SpacePlanogramFields.Name).ToString + ".PDF";
        SpacePlanning.PrintUsingLayout(@"C:\Program Files (x86)\JDA\Intactix\Intactix Knowledge Base\Layout\Front View.psy", SpaceSelection.CurrentPlanogram, "JDA PDF Writer", sFilePath);
        SpacePlanning.SaveScriptDocument(sName, Plano.GetPlanogramField(SpacePlanogramFields.Name).ToString + "PDF DOCUMENT", "PDF Document", "application/pdf", sFilePath);
        iCount = iCount + 1;
        }
    }
catch (Exception ex)
    {
    SpacePlanning.AddToMessageList(1000, ex.StackTrace, MessageSeverity.Error);
    SpacePlanning.AddToMessageList(1000, ex.Message, MessageSeverity.Error);
    }

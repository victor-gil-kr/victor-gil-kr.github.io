SwapSection

This command enables you to replace an individual section of a specified planogram with a section of another planogram.

Syntax

Visual Basic

Copy 
SwapSection ( _
    PlanogramKey As String, _
    SwapPlanogramKey As String, _
    CurrentFixture As String, _
    CurrentSectionX As Double, _
    CurrentSectionY As Double _
)

C#

Copy 
SwapSection(string PlanogramKey, string SwapPlanogramKey, [string CurrentFixture = null], [double CurrentSectionX = 0], [double CurrentSectionY = 0])

Parameters

Parameter

Description
PlanogramKey

String. The ID or Name for the planogram being added, modified, or deleted. When modifying or deleting planograms, or sections of planograms, both fields are searched for the specified value. The first planogram or section found matching the specified value is modified or deleted, unless optional FixtureName or location parameters are also specified. When adding a planogram from a planogram library to a project, the value specified must match the primary key (ID or Name) for the planogram library. Otherwise, the planogram is not found.
SwapPlanogramKey

The ID or Name of the planogram replacing the section being swapped. Both fields are searched for the specified value.
CurrentFixtureName (optional)

String. The name of the fixture on which a position or section currently exists. If omitted when swapping positions or sections, the command swaps the first encountered position or section for the specified product or planogram.

CurrentSectionX (optional)

Double. The optional horizontal location for the section. This parameter enables you to specify the exact location of the section, in case sections of the planogram exist in multiple locations on the floorplan. If an invalid or incorrect coordinate is specified, the section is not found, so it cannot be modified, moved, deleted, or swapped.

CurrentSectionY (optional)

Double. The optional vertical location for the section, as seen in 2D view. This parameter enables you to specify the exact location of the section, in case sections of the planogram exist in multiple locations on the floorplan. If an invalid or incorrect coordinate is specified, the section is not found, so it cannot be modified, moved, deleted, or swapped.

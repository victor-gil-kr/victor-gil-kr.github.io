Space Automation Professional OnLine Expert

Version 2021.1.2.0

Overview

Space Automation Professional enables you to use advanced tools and industry programming standards to develop scripts that automate Space Planning and Floor Planning tasks. Space Automation Professional provides Space Planning, Floor Planning, and Category Knowledge Base commands and helpers that you can use with the library of methods and functions available in Microsoft Visual Studio. These tools enable developers to build the necessary automation scripts to update and maintain objects on planograms and floorplans.

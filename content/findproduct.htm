FindProduct

This command enables you to determine if a specific product exists in the active project. If the product is found, the command returns a reference to the product, without selecting the product. If the product is not found, the commands returns a NULL value.

Note: If you want to locate and select a product in the active project, use the SelectProduct command instead.

Syntax

Visual Basic

Copy 
FindProduct ( _
    ProductKey As String, _    
)

C#

Copy 
Space.Product  FindProduct(string ProductKey)

Parameters

Parameter

Description
ProductKey

String. The ID or UPC for the product to be found. The first product found that matches the specified value is returned. If the matching key is set to ID+UPC, you should identify and specify both values, in the following format: "ID=12345;UPC=123456789".

Example: Find a product

Classic style scripting

Visual Basic

Copy 
Dim Prod As Space.Product
SpacePlanning.OpenProjectFile("C:\Program Files (x86)\JDA\Intactix\Samples\Space Planning Planograms\Sample - Coffee Makers.psa")
    Prod = SpacePlanning.FindProduct("07800801950")
        If Not (Prod Is Nothing) Then
            Prod.SetProductField(SpaceProductFields.AbbrevName, "Modified")
        End If

C#

Copy 
Space.Product Prod;
SpacePlanning.OpenProjectFile(@"C:\Program Files (x86)\JDA\Intactix\Samples\Space Planning Planograms\Sample - Coffee Makers.psa");
Prod = SpacePlanning.FindProduct("07800801950");
if (Prod != null)
Prod.SetProductField(SpaceProductFields.AbbrevName, "Modified");

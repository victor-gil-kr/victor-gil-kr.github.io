Code snippet examples

Some tasks previously supported by commands in legacy Space Automation do not require explicit commands in Space Automation Professional. Instead, code snippets can be used to perform these tasks. The following code snippets are available:

  • GetAndSetSpaceFieldValuesForAllRecords: Used to retrieve and set values for records in Space Planning.
  • GetAndSetFloorFieldValuesForAllRecords: Used to retrieve and set values for records in Floor Planning.
  • SetSpaceFieldValueForAllRecords: Used to set values for records in Space Planning.
  • SetFloorFieldValueForAllRecords: Used to set values for records in Floor Planning.
  • ExecuteDatabaseQuery: Used to retrieve data from a Category Knowledge Base database.

Insert a code snippet

  1. Right-click the script window and select Insert Snippet. The snippet menu is displayed.
  2. From the snippet menu, select JDA.Intactix.Automation.CodeSnippets. A submenu is displayed.
  3. From the submenu, select the appropriate option (Database, FloorPlanning, or SpacePlanning). A submenu is displayed.
  4. From the submenu, select the code snippet to be inserted. The snippet is added to the active script.

Note: You can enter the name of the code snippet and then press Tab to insert the snippet into the script:

  • GetAndSetSpaceFieldValuesForAllRecords
  • GetAndSetFloorFieldValuesForAllRecords
  • SetSpaceFieldValueForAllRecords
  • SetFloorFieldValueForAllRecords
  • ExecuteDatabaseQuery

You can also enter the first letters of the snippet name followed by a question mark (?) and then press Tab to display all snippets that begin with those letters.

Example: Get and set field values for Space Planning records

Classic style scripting

Copy 
' This script opens a sample project, selects objects on the planogram, and then modifies the selected objects.
' Open a project.
SpacePlanning.OpenProjectFile("C:\Program Files (x86)\JDA\Intactix\Samples\Space Planning Planograms\Sample - Merchandising Styles.psa")
' Update the Date Modified field with the current system date.
SpacePlanning.SetPlanogramField("12' Position Merchandising", SpacePlanogramFields.DateModified, Now)
' This section uses Select commands to select the object to be processed, and Set Field commands to modify the object properties.
SpacePlanning.SetSegmentField(1, SpaceSegmentFields.Height, 100)
SpacePlanning.SetDrawingField("Case", SpaceDrawingFields.BackColor, RGB(241, 196, 15))
SpacePlanning.SelectFixture("2")
SpacePlanning.SetPositionField("0002222222222", SpacePositionFields.HFacings, 5, "2")
SpacePlanning.SelectSegment(1, True)
SpacePlanning.SelectPositionByLocationID(2, True)
' This section gets the active position, and then uses Instance support to modify the object properties.
Dim myPos As Space.Position
myPos = SpacePlanning.ActivePosition
myPos.Roll = 12
' This section uses object-specific Set Field commands to modify field values for each object type.
SpacePlanning.SetPlanogramField("12' Position Merchandising", SpacePlanogramFields.Color, RGB(241, 196, 15))
SpacePlanning.SetSegmentField(2, SpaceSegmentFields.Height, 45)
SpacePlanning.SetFixtureField("3", SpaceFixtureFields.Height, 5)
SpacePlanning.SetPerformanceField("0004444444444", SpacePerformanceFields.Desc1, "This is sample text")
SpacePlanning.SetDrawingField("The Green product ...", SpaceDrawingFields.String, "THIS IS NEW TEXT")
SpacePlanning.SetProductField("0004444444444", SpaceProductFields.UPC, Rnd(13).ToString())
' This section selects the positions to be processed, and then sets the color of the associated products.
SpacePlanning.SelectPosition("0002222222222")
SpacePlanning.SelectPosition("0003333333333")
For Each myObj As Object In SpacePlanning.ForSelectedObjects()
    SpacePlanning.SetProductField(CType(myObj, Space.Position).Product.ID, SpaceProductFields.Color, RGB(93, 109, 126))
Next
' This section selects a fixture using its Location ID, sets a Position field value, and then copies another value from one Position field into another for the positions on the selected fixture.
SpacePlanning.SelectFixtureByLocationID(1)
For Each myPos2 As Space.Position In SpacePlanning.ForFixturePositions()
    myPos2.HFacings = 2
    myPos2.Desc1 = myPos2.Product.Name
Next

Example: Set field values for all Space Planning records

Copy 
' This script opens a sample project and then removes data from products in the project.
' Open a project.
SpacePlanning.OpenProjectFile("C:\Program Files (x86)\JDA\Intactix\Samples\Space Planning Planograms\Sample - Coffee Makers.psa")
' Set various Product field values to 0.
For Each prod As Space.Product In SpacePlanning.ForProducts()
    prod.UnitMovement = 0
    prod.UnitCost = 0
    prod.Price = 0
Next

Example: Get and set field values for Floor Planning records

Copy 
' This script opens a sample project, selects objects on the floorplan, and then modifies the selected objects.
' Open the sample file.
FloorPlanning.OpenProjectFile("C:\Program Files (X86)\JDA\Intactix\Samples\Floor Planning Floorplans\Sample - Store 001.pfa")
' Select the floorplan in the project containing the objects to be processed.
FloorPlanning.SelectFloorplan("001 sample store")
' This section uses Select commands to select the object to be processed, and Set Field and SetFieldValue commands to modify object colors.
FloorPlanning.SetDepartmentField("GROCERY", FloorDepartmentFields.Color, RGB(255, 150, 120))
FloorPlanning.SetPlanogramField("001006", FloorPlanogramFields.Color, RGB(255, 0, 0))
FloorPlanning.SelectFixtureByLocationID(37)
FloorPlanning.SetFieldValue(FloorSetFieldObject.Fixture, FloorFixtureFields.Color, RGB(255, 0, 0))
FloorPlanning.SelectSectionByLocationID(16)
FloorPlanning.SetFieldValue(FloorSetFieldObject.Planogram, FloorPlanogramFields.Color, RGB(255, 0, 0))
' This section uses object-specific Set Field commands to modify field values for each object type.
FloorPlanning.SetFloorplanField("001 sample store", FloorplanFields.FloorColor, RGB(255, 120, 110))
FloorPlanning.SetDepartmentField("Snacks", FloorDepartmentFields.Color, RGB(255, 0, 0))
 FloorPlanning.SetFixtureField("0045(DAIRY)", FloorFixtureFields.Depth, 100)
FloorPlanning.SetPerformanceField("001006", FloorPerformanceFields.Desc1, "This is Sample Text")
FloorPlanning.SetSectionField("011152", FloorSectionFields.DepartmentOverride, "Grocery")
' This section uses a ForSelectedObjects loop to change the color of selected sections.
FloorPlanning.SelectSection("012159", , , , True)
FloorPlanning.SelectSection("012160", , , , False)
For Each selObject As Object In FloorPlanning.ForSelectedObjects()
    FloorPlanning.SetFieldValue(FloorSetFieldObject.Planogram, FloorPlanogramFields.Color, RGB(111, 140, 145))
Next
' This section copies the field value from one Planogram field to another.
For Each pog As Floor.Planogram In FloorPlanning.ForPlanograms
    pog.Value31 = pog.Value1
Next

Example: Execute a database query (SQL Server)

Copy 
' Add "Imports System.Data.SqlClient" in import section at the top of script
' Update the following section with the appropriate database details.
' Example:
'Dim Server As String = "(local)"
'Dim Database As String = "CKB_2016"
'Dim DBUsername As String = "sa"
'Dim DBPassword As String = "admin$123"

Dim Server As String = "Server Name"
Dim Database As String = "Database Name"
Dim DBUsername As String = "User Name"
Dim DBPassword As String = "Password"
' For Windows login, connectionstring = "Server= localhost; Database= employeedetails; Integrated Security=True;" 
Dim SQLConnectionString As String = String.Format("Server={0}; Database={1}; User Id={2}; Password={3}", Server, Database, DBUsername, DBPassword)
Dim planogramarray As New ArrayList()
' Create a database connection to execute a custom query on the SQL database. 
Try
    Using DBConnection As New SqlConnection(SQLConnectionString)
        DBConnection.Open()
        ' Update the following query as appropriate.
        Dim sqlString As String = "SELECT * from ix_spc_planogram where status1= 'live'"
        ' Create a dataset. 
        Dim myDataSet As DataSet = New DataSet()
        ' Create the data adapter.
        Dim SQLAdapter As SqlDataAdapter = New SqlDataAdapter(sqlString, DBConnection)
        'Fill the data adapter with the dataset.
        SQLAdapter.Fill(myDataSet)
        'Create a datatable.
        Dim planogramTable As DataTable = New DataTable()
        ' Initialize the datatable.
        planogramTable = myDataSet.Tables(0)
        'Add the DBkey (Note: The DBkey is a keyword, but this field name could be anything - not a CKB field.)
        For Each tblRow As DataRow In planogramTable.Rows
            planogramarray.Add(tblRow("DBkey"))
        Next
    End Using
    'After getting the list of suitable planogram DBkeys, open each of the planograms in Space Planning using its DBKey. 
    SpacePlanning.SelectDatabase(Database + "|" + Server + "|" + DBUsername)
    For Each dbkey As Integer In planogramarray
        SpacePlanning.OpenPlanogramFromDatabase(dbkey)
    Next
    Catch ex As Exception
    MsgBox(ex.Message)
End Try
